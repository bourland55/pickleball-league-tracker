# Screenshots

This folder holds the screenshots referenced by the README files.

## Recommended captures

Replace any of these with your own screenshots from your branded install. **Sanitize first** — blur or rename real player names if you don't want them in a public repo.

| Filename | What to capture |
|---|---|
| `01-this-week.png` | "This Week" tab with a week set up (players assigned to courts) |
| `02-courts.png` | "Courts" tab showing live scoring, mid-session |
| `03-leaderboard.png` | "Leaderboard" tab with the current week's standings |
| `04-season.png` | "Season" tab — pick the Standings sub-tab |
| `05-admin.png` | "Admin" tab logged in, showing the roster |
| `06-cyberpunk.png` | Same as 01 or 02, but with the cyberpunk theme on (tap the theme toggle twice) |
| `hero.png` *(optional)* | A 5-up collage of the others for the README hero image |

## Capture tips

- Use a **mobile-portrait** browser window (~390×844 — Safari iPhone emulation in DevTools works great). The app's designed for phones.
- Use the **default brand color** (`#22c55e`) or pick something photogenic.
- Show **realistic but generic data** — "Player 1", "Player 2", etc., is safest for public sharing.
- Save as PNG, ideally 800px wide max to keep the repo small.

## Sample logo

The default `assets/logo.svg` is a simple placeholder. Replace it with your league's actual logo (PNG or SVG). The header crops to ~36px tall and scales the width automatically.
