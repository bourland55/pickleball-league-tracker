Run a Weekly Tournament. No Coding Required.

A complete, customizable scoresheet app for your weekly pickleball (or ping-pong, or doubles tennis) league. Live scoring on phones. Season standings. Audit trail. Three themes. Zero build step.

▸ Quick start with Claude Code
"let's get started"
Open this folder in Claude Code and type the line above. The setup wizard walks you through naming your league, picking a brand color, uploading a logo, choosing solo or multi-user mode, and deploying — about 10-15 minutes, no manual editing required.

02What You Get

📅 This Week

Set up the week's roster, assign players to courts in groups of 4, lock the schedule. Live as players arrive.

🏓 Live Court Scoring

Score entry per game, per series. Auto-detects winners, "PICKLED" 11-0 shutouts, in-progress vs. complete.

📊 Leaderboard

Per-week standings with rank deltas, game log, edit audit history. Filter by completed week.

🏆 Season Standings

Cumulative wins, points, absence-adjusted totals. Week-by-week heatmap. Absence tracking with penalty rules.

🛠 Admin Tools

Roster management, week setup wizard, score audit log with reason-tagged edits, league settings.

🎨 Three Themes

Dark, light, and a glowing cyberpunk mode. One-click toggle, persists across sessions.

📱 Mobile-first

Designed for entering scores from a phone next to the court. Big tap targets, no fiddling.

🔄 Live Multi-user Sync

Optional Firebase mode means everyone sees scores update in real time on their own device.

🏠 Or Just Solo

No setup, no internet, no Firebase. Just open index.html and play. Browser storage handles it.

04Manual Install

If you'd rather skip Claude Code:

  1. Copy config.example.js → config.js
  2. Edit the values in config.js (league name, brand color, logo, admin password, storage mode)
  3. Solo mode: open index.html. Done.
  4. Multi-user mode: follow FIREBASE_SETUP (or the .md version), then open index.html.

Full guide: docs/INSTALL.md

05Customize

One file controls almost everything: config.js.

window.LEAGUE_CONFIG = {
    leagueName:  "Tuesday Night Pickle Crew",
    sport:       "Pickleball",
    logoSrc:     "assets/logo.png",
    brandColor:  "#22c55e",
    storageMode: "local",
    adminPassword: "changeme",
    // ... see config.example.js for the full list
};
Quick palette swaps

Try #3b82f6 (blue) · #8b5cf6 (purple) · #f59e0b (orange) · #ef4444 (red) · #14b8a6 (teal) · #ec4899 (pink). Or any 6-digit hex.

For deeper changes (new tabs, custom scoring, additional features), open this folder in Claude Code and ask. See docs/PROMPTS.md for ready-to-paste prompts.

Heads up — Firebase mode

After deploying, add your deployment URL to Firebase Console → Authentication → Settings → Authorized domains. Otherwise users will see "permission denied" errors.

08Tech Stack

  • Pure HTML + JavaScript — no build step, no framework runtime, no npm install
  • Tailwind CSS via CDN for styling
  • Firebase Web SDK v9 (compat) — optional, only used in multi-user mode
  • Total weight: ~70KB compressed. Loads on cellular in <2 seconds.

The app works fully offline in solo mode. In multi-user mode it needs an internet connection to sync with Firebase.

09Sport Scope

Out of the box, this template is built around pickleball-style mechanics: 11-point games, 4-player doubles with a 3-series rotation. Easy to rename for any 2v2 sport with similar rules (ping-pong doubles, padel, doubles tennis to 11).

For different formats (singles, 3v3, custom rotations), you'd need code changes — open the folder in Claude Code and describe what you want.
